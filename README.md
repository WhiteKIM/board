## 개요
스프링부트와 mariadb를 이용한 간단한 게시판입니다.
## 구현 기능
1. 로그인 -> 추후 스프링 시큐리티로 변경될 예정
2. 회원가입 -> 위의 스프링 시큐리티 적용과정에서 변경
3. 글 작성 -> 완료
4. 글 수정 -> 완료
5. 디자인 -> css를 이용할 예정
